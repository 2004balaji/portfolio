# Personal Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Dark theme with glassmorphism design
- Smooth scrolling navigation
- Animated section transitions on scroll
- Responsive design for all devices
- Professional layout with premium typography
- Download resume functionality

## Technologies Used

- HTML5
- CSS3 (with glassmorphism effects)
- JavaScript (ES6+)
- Google Fonts (Inter)

## Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── style.css          # Stylesheet with responsive design
├── script.js          # JavaScript for animations and interactions
├── resume.pdf         # Resume file
└── WhatsApp Image 2026-01-23 at 12.36.31 AM.jpeg  # Profile image
```

## Deployment on Vercel

1. **Connect Repository**: Link your GitHub repository to Vercel
2. **Auto-Deployment**: Vercel will automatically detect and deploy the static site
3. **Custom Domain** (optional): Add your custom domain in Vercel settings

### Manual Deployment Steps:

1. Install Vercel CLI: `npm install -g vercel`
2. Login: `vercel login`
3. Deploy: `vercel` (from project root)
4. Follow the prompts to set up the project

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. For live reload, use a local server like:
   ```bash
   python -m http.server 8000
   # or
   npx serve
   ```

## Customization

- Update personal information in `index.html`
- Modify colors and styles in `style.css`
- Resume file is already added as `resume.pdf`
- Profile image is already added as `WhatsApp Image 2026-01-23 at 12.36.31 AM.jpeg`
